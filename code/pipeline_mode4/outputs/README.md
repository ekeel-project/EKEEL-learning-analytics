# Mode 4 outputs

This folder is intentionally empty in the deliverable package.

It will be populated when you run the Mode 4 pipeline. Typical artifacts include:
- `Report/Mode4Features.csv` (feature table)
- `Report/report.html` (summary report)
- `Report/ProcessedLogs/` (per-session cleaned CSV logs for sequence modeling)
- `Report/Predictive/` (Random Forest and HMM outputs)
