## Overview
**This repository contains the EKEEL learning analytics reference implementation for analysing interaction and usage data collected during the experimentation with the EKEEL Video Augmentation project prototype in evaluation settings, using Operation mode 2 and operation mode 4. It supports experimental use and reproducible evaluation.**

- **Project:** EKEEL — PRIN 2022 PNRR
- **Deliverable:** **D4.1** — Learning analytics model
- **Repository:** `ekeel-project/EKEEL_learning_analytics`
- **Website:** https://ekeel.dibris.unige.it/

## What’s in this repository
- `code/` — Analysis code (scripts/modules) and notebooks for the learning analytics pipeline.
- `docs/` — Technical documentation, usage notes, and reporting material.

### Requirements
**OS**
- Windows / macOS / Linux

**Runtime / tooling**
- Python (recommended: 3.10+)
- Jupyter Notebook / JupyterLab (if notebooks are provided)



### Installation
```bash
git clone https://github.com/ekeel-project/EKEEL-learning-analytics.git
cd EKEEL-learning-analytics
```

**Quick start**
- Open the repository notebooks and run the analysis end-to-end
